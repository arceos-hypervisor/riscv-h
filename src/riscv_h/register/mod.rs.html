1//! RISC-V Hypervisor Extension Registers
2//!
3//! This module provides access to Control and Status Registers (CSRs) defined
4//! in the RISC-V Hypervisor Extension. These registers enable virtualization
5//! support by providing hypervisor-level control and virtual machine state management.
6//!
7//! ## Register Categories
8//!
9//! ### Hypervisor Control Registers
10//! These registers control hypervisor behavior and guest execution:
11//! - `hstatus` - Hypervisor status register
12//! - `hedeleg` - Hypervisor exception delegation register
13//! - `hideleg` - Hypervisor interrupt delegation register
14//! - `hie` - Hypervisor interrupt enable register
15//! - `hip` - Hypervisor interrupt pending register
16//! - `hvip` - Hypervisor virtual interrupt pending register
17//! - `hcounteren` - Hypervisor counter enable register
18//! - `hgatp` - Hypervisor guest address translation and protection register
19//! - `htimedelta` - Hypervisor time delta register
20//! - `htimedeltah` - Hypervisor time delta high register
21//! - `htval` - Hypervisor trap value register
22//! - `htinst` - Hypervisor trap instruction register
23//! - `hgeie` - Hypervisor guest external interrupt enable register
24//! - `hgeip` - Hypervisor guest external interrupt pending register
25//!
26//! ### Virtual Supervisor Registers
27//! These registers manage virtual machine supervisor-level state:
28//! - `vsstatus` - Virtual supervisor status register
29//! - `vsie` - Virtual supervisor interrupt enable register
30//! - `vsip` - Virtual supervisor interrupt pending register
31//! - `vsepc` - Virtual supervisor exception program counter
32//! - `vscause` - Virtual supervisor cause register
33//! - `vstval` - Virtual supervisor trap value register
34//! - `vstvec` - Virtual supervisor trap vector register
35//! - `vsscratch` - Virtual supervisor scratch register
36//! - `vsatp` - Virtual supervisor address translation and protection register

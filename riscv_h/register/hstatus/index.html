Hypervisor status register Hypervisor Status Register.

The hstatus register provides status and control information for the hypervisor. It contains fields that control virtual machine execution, interrupt handling, and memory management in virtualized environments.

This register is central to RISC-V hypervisor operation and includes fields such as:

  • Virtual machine privilege and execution state
  • Guest virtual address translation controls
  • Virtual interrupt management
  • Hypervisor user mode support

Hypervisor exception delegation register
Hypervisor Exception Delegation Register.

The hedeleg register controls which exceptions are delegated from HS-mode to VS-mode. When a bit is set in this register, the corresponding exception will trap to VS-mode instead of HS-mode when occurring in VS-mode or VU-mode.

This register enables efficient virtualization by allowing guests to handle common exceptions (like page faults) directly without hypervisor intervention. Exception codes correspond to standard RISC-V exception cause values.
